# korea100studio Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `korea100studio`, a standalone Agent Skill that renders any process (lanes × stages × nodes × edges) into a vertical swimlane board (SVG, optional PNG), audits its composition quality, and produces a stage-ordered reveal animation — usable unchanged in Claude Code and Codex.

**Architecture:** Pure Node.js ESM. A generic `board-v1` JSON schema feeds a profile-driven layout+SVG core ported and generalized from korea100's `generate-process-article-image.mjs` (including the gutter-routing fix). Korea-specific look lives only in a `gov` profile. SVG is always produced; PNG is optional via a detected rasterizer. `lib/` exposes a programmatic API so korea100 can later soft-replace its renderer (Phase 2, out of scope here). korea100 is NOT modified by this plan.

**Tech Stack:** Node.js ≥20 (ESM), `node:test`, `ajv` (schema validation), optional `rsvg-convert`/`cairosvg` at runtime. No native build deps.

**Source of truth for the port:** `~/korea100/web/scripts/generate-process-article-image.mjs` and `~/korea100/web/src/lib/process-layout.mjs` (read-only reference; do not edit korea100).

import fs from "node:fs";

import { fileURLToPath } from "node:url";

const __dirname = path.dirname(fileURLToPath(import.meta.url));
const schema = JSON.parse(
  fs.readFileSync(path.join(__dirname, "../../schemas/board-v1.schema.json"), "utf8")

const ajv = new Ajv({ allErrors: true, strict: false });

## Task 9: Optional rasterizer

**Files:**

No native dep. Detect `rsvg-convert` then `cairosvg` on PATH; if neither, return `{ ok: false, reason }`.

- [ ] **Step 1: Write `scripts/lib/rasterize.mjs`**

```js
import { execFileSync, spawnSync } from "node:child_process";
import fs from "node:fs";

function has(bin) {
  return spawnSync("which", [bin]).status === 0;
}

// Rasterize an SVG file to PNG. Returns { ok, tool } or { ok:false, reason }.
export function rasterize(svgPath, pngPath, width = 1800) {
  if (has("rsvg-convert")) {
    execFileSync("rsvg-convert", ["-w", String(width), svgPath, "-o", pngPath]);
    return { ok: true, tool: "rsvg-convert" };
  }
  if (has("cairosvg")) {
    execFileSync("cairosvg", [svgPath, "-o", pngPath, "--output-width", String(width)]);
    return { ok: true, tool: "cairosvg" };
  }
  return { ok: false, reason: "no rasterizer (install librsvg or cairosvg for PNG)" };
}

export function rasterizerAvailable() {
  return has("rsvg-convert") || has("cairosvg");
}
```
